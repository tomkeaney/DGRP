This website is a companion page to our main page Becca’s shiny app here. Here you can find all of the code used to produce the analysis shown on the main page, with step by step annotation.

We document three different steps in our analysis:

  1. Data collation, which shows how we collected the data and the data cleaning process.

  2. Calculating genetic trait correlations, which details how we estimated \(r_g\).

  3. Calculating ‘SNP-heritability’, which details how we estimated heritability for each trait.

  4. Running genome-wide association tests, which describes how GWA tests were run and shows some summary statistics.

  5. Cross phenotype meta analysis, which details our CPMA analysis seraching for pleiotopic variants.

We also provide a guide to uploading your own data, so that this resource can remain current.
